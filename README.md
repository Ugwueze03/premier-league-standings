# Premier League Standings Analysis: 2023-2026


End-to-end project transforming raw match results into final league tables for 3 Premier League seasons.

### Objective

Use Excel and SQL to aggregate match-level data and generate accurate final standings, including points, goal difference, and final position for seasons 2023/2024, 2024/2025, and 2025/2026.

### Tech Stack

- *Excel*: Data validation, QA, and CSV export
  
- *SQL*: Data aggregation, CASE statements for wins/draws/losses, ranking with ROW_NUMBER
 
- *GitHub*: Version control and project documentation
